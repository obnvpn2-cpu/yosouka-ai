## 🚀 起動方法

### Step 1: ファイル配置

```bash
cd ~/デスクトップ/repo/keiba-yosoka-ai

# frontendディレクトリを作成
mkdir -p frontend

# ダウンロードしたindex.htmlを配置
cp ~/Downloads/index.html frontend/
```

### Step 2: APIサーバーを起動

**ターミナル1**:
```bash
cd ~/デスクトップ/repo/keiba-yosoka-ai

# APIサーバー起動
python backend/api/api.py
```

### Step 3: フロントエンドを開く

**ブラウザで開く**:
```bash
# frontend/index.htmlをブラウザで開く
start frontend/index.html  # Windows
open frontend/index.html   # Mac
xdg-open frontend/index.html  # Linux
```

または、ブラウザから直接開く：
```
file:///C:/Users/[ユーザー名]/デスクトップ/repo/keiba-yosoka-ai/frontend/index.html
```

## 🐛 トラブルシューティング

### エラー: "選択肢の取得に失敗しました"

**原因**: APIサーバーが起動していない

**解決策**:
```bash
# APIサーバーを起動
python backend/api/api.py
```

### エラー: "検索に失敗しました"

**原因**: CORS設定の問題、またはAPIエラー

**解決策**:
1. APIサーバーのログを確認
2. ブラウザのコンソール（F12）を確認

### 結果が表示されない

**原因**: 条件が厳しすぎる

**解決策**:
- 条件を緩める（「指定なし」を増やす）
- 距離の選択を増やす
